Preguntas del laboratorio

ODBC (Open Data Base Conectivity) es una API para el acceso a bases de datos con SQL independientemente del DBMS que se utilice. Las aplicaciones utilizan funciones del ODBC, que a su vez usa drivers para comunicarse con la base de datos.

SQL injection es la forma de hackeo de sitios web más común. Involucra la introducción de declaraciones en SQL en los campos de entrada de una aplicación web que son usados directamente para hacer consultas o modificaciones a la base de datos. Cualquier declaración escrita en el input pasará a formar parte de la query y podrá hacer cambios en la información de la aplicación.

Es posible protegerse contra SQL injection usando prepare statements desde php. Para esto, se crea una plantilla para el statement a la que se le añaden después los parámetros, especificando su formato. El SQL injection también pued evitarse a través de validaciones en el lado del servidor.
