1. Una base de datos es un conjunto de archivos que representan entidades modelables e interrelacionadas a través de sus atributos relevantes. Suelen utilizarse para almacenar grandes cantidades de información que está organizada para ser accedida rápidamente por varios programas. Está integrada con un sistema administrador de bases de datos, que funge como interfaz entre la información y los sistemas (concurrentes o no) que la utilizan, facilitando su uso.

  2. Es conveniente utilizar bases de datos cuando se quiere almacenar una gran cantidad de información que está interrelacionada lógicamente y será accedida por múltiples aplicaciones.

  3. Un sistema administrador de bases de datos (o DBMS) es un sistema que funge como interfaz entre la base de datos y otros usuarios o programas. Su principal función es ser una herramienta centralizada y efectiva para la consulta, creación, borrado y almacenamiento de datos, proveyendo seguridad, información íntegra, funcionalidades de concurrencia, seguridad y uniformidad en los procesos administrativos.

  4. Funciones de un DBMS:

    • Administrar los datos
    • Constituye el motor de una base de datos, permitiendo el acceso, modificación y control sobre la información.
    • Provee el esquema de la base de datos, lo que define la estructura lógica de los datos almacenados.
    • Registro de actividades para la generación de controles.
    • Monitoreo de desempeño para facilitar a los administradores la detección de problemas
    • Recuperación y respaldo de la información, en caso de que esta se dañe u ocurra algún incidente en el sitio principal de la base de datos.

  5. Considero que una base de datos habría sido de gran utilidad cuando hice una página que cambiaba de lenguaje obteniendo información de un archivo XML. El XML era complicado de editar debido a su tamaño y hacía fallar todas las peticiones posteriores a un error de formato, por lo que alamcenar los datos en varias tablas que estuviesen integradas con un software especial para edición y consulta habría facilitado mucho las cosas y recortado bastante el tiempo de desarrollo del proyecto
