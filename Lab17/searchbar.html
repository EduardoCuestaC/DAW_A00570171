AJAX permite la creación de páginas web con un mayor nivel de interactividad y mejora mucho la experiencia de usuario en la utilización de contenido dinámicamente generado desde el servidor.

      Con AJAX es posible utilizar el misimo código para validar tanto en el cliente como en el servidor. Las páginas que utilizan AJAX tienen las mismas vulnerabilidades que las páginas que no lo usan.

      JSON (Javascript Object Notation) es una sintaxis para el envío de información entre un cliente y un servidor que puede traducirse fácilmente a objetos de Javascript o a otros tipos de arreglos.
